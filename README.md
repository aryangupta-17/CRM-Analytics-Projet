# CRM-Analytics-Projet

 CRM Analytics & Sales Performance Dashboard
🚀 Overview

This project focuses on analyzing CRM datasets to uncover actionable insights related to sales performance, customer behavior, and lead conversion efficiency.
The goal is to transform raw data into meaningful visualizations and support data-driven decision-making.

🎯 Objectives
Analyze sales pipeline performance and opportunity trends
Evaluate lead generation and conversion efficiency
Identify high-performing industries and channels
Detect data quality issues and reporting gaps
Support strategic business decisions through insights

🛠️ Tools & Technologies
Excel – Data cleaning, preprocessing, validation
MySQL – Data extraction, transformation, querying
Tableau – Interactive dashboards and storytelling
Power BI – KPI tracking and business reporting

📂 Dataset
CRM opportunity and lead datasets
~10,000+ records analyzed
Includes fields like revenue, industry, lead source, conversion status, and opportunity stages

📊 Key Features

🔹 Opportunity Analysis
Revenue and pipeline performance tracking
Win/Loss and conversion rate analysis
Industry-wise opportunity distribution
Expected vs forecast insights
🔹 Lead Analysis
Lead source performance comparison
Industry-wise lead distribution
Lead-to-conversion funnel analysis
Conversion rate tracking (~11%)

📈 Key Insights
Identified 33% revenue dependency on a single industry, highlighting diversification risk
Detected low conversion rate (~11%), indicating funnel inefficiencies
Observed post-2020 decline in deal closures, suggesting process gaps
Found 10% missing/uncategorized data, impacting reporting accuracy

⚠️ Challenges Faced
Handling missing and inconsistent data entries
Managing unstructured categories (uncategorized opportunities)
Low conversion rates affecting pipeline performance
Ensuring data consistency across multiple sources

💡 Solutions Implemented
Performed data cleaning and validation using Excel and SQL
Structured datasets for consistent reporting
Built interactive dashboards for better visibility
Highlighted key issues with actionable recommendations

📊 Dashboard Highlights
3+ interactive dashboards with dynamic filters
10+ KPIs tracked (Revenue, Conversion Rate, Win Rate, etc.)
Drill-down capabilities for detailed analysis
Real-time performance monitoring

🎯 Outcome
Improved reporting efficiency by ~30%
Reduced manual analysis effort by ~40%
Enabled faster and more accurate decision-making
Delivered actionable insights for business growth

📌 Conclusion

This project demonstrates strong capabilities in data analysis, SQL querying, dashboard development, and business insight generation, making it highly relevant for data analyst roles.
